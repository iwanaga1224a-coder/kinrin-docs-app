# 近隣説明会 書類生成アプリ

建設工事の近隣説明会に必要な届出書類を、フォーム入力だけで一括生成するWebアプリです。

## 生成できる書類

| # | 書類 | 形式 |
|:--|:-----|:-----|
| 1 | 近隣説明範囲図（地図付き） | Word (.docx) |
| 2 | 標識設置届 | Word (.docx) |
| 3 | 近隣説明報告書 | Word (.docx) |
| 4 | 工事のお知らせ（住民配布チラシ） | Word (.docx) |

## セットアップ

```bash
pip install -r requirements.txt
```

Chrome ブラウザがインストールされている必要があります（地図のスクリーンショット生成に使用）。

## 起動

```bash
streamlit run app.py
```

ブラウザで `http://localhost:8501` が自動的に開きます。

## 使い方

1. フォームに工事情報を入力（最低限「工事名称」と「工事場所」だけでOK）
2. 「書類を生成する」ボタンをクリック
3. ZIPファイルでWord書類4点＋地図PNGをダウンロード

## 注意事項

- 生成した書類は叩き台です。届出前に必ず管轄の行政窓口で様式・記載内容をご確認ください。
- 東京都中高層建築物紛争予防条例の様式をベースにしています。
